Passo a passo para realizar a sincronização 

1° Passo 
- Copie o codigo do Script e salve no seu editor de texto

2° Passo 
- Salve dessa forma "script.sh"

3° Passo 
- Abra o seu terminal CTRL + ALT + T navegue ate o local que vc salvou o Script

4° Passo 
- no terminal de o comando "chmod +x script.sh" esse codigo vai deixar o script com permissão de execução

5° Passo
- Rode no terminal o comando ./script.sh

6° Passo 
- Coloque seu nome de usuario

7° Passo 
- Coloque seu email de acesso ao GitHub

8° Passo 
- Copie a chave gerada pelo terminal abaixo da frase "Sua configuração terminou. Copie a chave abaixo e coloque no GitHub:"

9° Passo 
- Abra o seu Git Hub, va em perfil - configurações - SSH And GPG keys

10° Passo 
- Crie uma nova chave SSH, De um título para ela e cole o codigo gerado no passo no 8° passo no campo key

11°
- Clique no botão "add SSH key"
